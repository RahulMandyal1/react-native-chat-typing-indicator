# Contributing

Thanks for contributing! To keep things efficient, please follow these guidelines.

## Reporting Issues
**Issues must be reproducible.**
When reporting a bug, please include:
1. **Description**: What went wrong?
2. **Minimal Reproduction**: A small code snippet or repo showing the error.


## Pull Requests
- Run `npm run typecheck` and `npm run format` before submitting.
- Use Conventional Commits (e.g., `fix: ...`, `feat: ...`).
